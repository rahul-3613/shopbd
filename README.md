## ✨ Features

- 🎠 **Auto Hero Slider** — 3-slide banner with dot & arrow controls
- ⚡ **Flash Sale Section** — Live countdown timer with progress bars
- 🗂️ **Category Grid** — 8 product categories with icons
- 🛒 **Cart Sidebar** — Add/remove items, quantity control, live total
- ❤️ **Wishlist** — Toggle products, badge updates
- 🔍 **Live Search** — Suggestions dropdown on keypress
- 🎛️ **Product Filter Tabs** — Filter by Electronics, Fashion, Home, Books
- 📱 **Fully Responsive** — Works on mobile, tablet & desktop
- 💌 **Newsletter** — Email validation with toast feedback
- 🎨 **Smooth Animations** — Scroll reveal, card hover, counter animation
- 🧾 **Mega Menu** — Full category navigation dropdown
- 📣 **Toast Notifications** — Real-time user feedback on all actions

## 🚀 Getting Started

### Run Locally

```bash
# 1. Clone the repository
git clone [https://github.com/your-username/shopbd.git](https://rahul-3613.github.io/shopbd/)

# 2. Open the folder
cd shopbd

# 3. Open in browser (or use VS Code Live Server)
open index.html
```

> 💡 **Tip:** Use the [Live Server]([https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer](https://rahul-3613.github.io/shopbd/)) extension in VS Code for the best experience.
